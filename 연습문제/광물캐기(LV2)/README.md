# 문제 제목 
- 광물 캐기

# 문제 난이도
- Level

# 문제 링크
- https://school.programmers.co.kr/learn/courses/30/lessons/172927

# 문제 핵심
- 최소한의 피로도 찾기
- 광물은 순서대로만 캔다.
- 곡괭이 하나 쓰면 중간에 바꾸지 못한다
- 힘든 구간에는 좋은 곡괭이를 써야한다.

# 제한 사항
- 곡괭이의 개수를 나타는 정수 배열 : picks
  0 <= dia,iron,stone <= 5
- 광물의 순서를 나타내는 문자열 배열 : minerals
  5 <= minerals 의 길이 <=50

# 문제 유형 
- 그리디

# 풀이순서
  1. 광물을 5개씩 묶음
