# 문제 제목 
완전범죄

# 문제 링크
https://school.programmers.co.kr/learn/courses/30/lessons/389480

# 제한사항
1 <= info의 길이 <= 40

# 문제 유형
- 1. 완전탐색 : 2^40 = 1조 == 불가능
- 2. DP     : A가 훔친다 or B가 훔친다.
(문제에 최솟값, 최댓값, 최적 이런 표현이 나오면 최적화문제로 DP확률이 매우 높다)

dp[a] = A 흔적이 a일 때 가능한 최소 B 흔적

# 시간복잡도
DP 크기 = [A흔적 최대 = N <= 120] = 120
40 * 120 = 4800
