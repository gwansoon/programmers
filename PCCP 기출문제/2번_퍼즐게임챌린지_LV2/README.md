# 문제 유형
- 이분탐색 binary_search
- level ↑  ⇒ total_time ↓
- level ↓  ⇒ total_time ↑
숙련도 level이 변할 때 총 소요 시간의 변화:
- level이 증가하면
    → 틀리는 횟수 감소
    → 총 시간 감소 또는 동일
- level이 감소하면
    → 틀리는 횟수 증가
    → 총 시간 증가
단조성이 있기 때문에 이분 탐색 가능함.
